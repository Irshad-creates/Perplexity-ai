# PerplexCity AI

A modern AI-powered SaaS-style web application inspired by Perplexity AI, built using the MERN stack with real-time chat, AI integrations, intelligent search, and AI-generated email capabilities.

---

# Features

## AI Chat System

* Real-time AI conversations
* Persistent chat history
* Dynamic chat titles
* Markdown-rendered AI responses
* Auto-growing chat input
* Responsive chat interface
* Recent chat management
* Delete chats functionality

---

## AI Email Assistant

* Generate professional emails using AI
* Send emails with Gmail OAuth2
* Prompt-based email writing
* Modern email workspace UI
* AI-powered subject/content generation

---

## Authentication

* User registration & login
* JWT authentication
* Protected routes
* Persistent sessions
* Logout functionality
* Toast notifications for auth actions

---

# UI / UX

* Modern dark SaaS interface
* Fully responsive design
* Mobile sidebar navigation
* Glassmorphism effects
* Smooth hover animations
* Custom toast notifications
* Dynamic avatars using user initials
